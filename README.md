# Username Generator API
The usernmae generator API is an API which returns a random username containing a prefix, a suffix, and a random number from 1 to 1000 at the end of the username. It can be used for generating usernames, entity names in a simulation, etc.

# How to use
To utilize the username generator API, use your application to access https://www.justanothercodetetser.github.io/username-generator-api, and the website will return a plaintext string containing the username.
